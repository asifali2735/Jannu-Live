# **App Name**: ConnectSphere

## Core Features:

- User Authentication: Secure user registration and login.
- Content Posting: Allow users to upload short videos and pictures.
- Newsfeed: Display content from followed users in a personalized newsfeed.
- Commenting and Liking: Enable users to comment on and like posts. Optionally offer secret like function
- Live Chat: Implement real-time text-based chat functionality.
- Group Video Call: Support live video calls with multiple participants.
- Monetization via Ads: Integrate advertising to generate revenue.

## Style Guidelines:

- Primary color: Light blue (#55acee) to represent connectivity and calmness.
- Background color: Off-white (#f5f8fa) to create a clean and modern interface.
- Accent color: Orange (#ff8533) to highlight interactive elements and calls to action.
- Body font: 'PT Sans', a versatile sans-serif for readability and a modern feel.
- Headline font: 'Space Grotesk', paired with 'PT Sans' for the body. Space Grotesk provides a techy, scientific feel, in headlines and short bits of text.
- Use minimalist and recognizable icons for easy navigation.
- Subtle transitions and animations to provide a smooth user experience.
